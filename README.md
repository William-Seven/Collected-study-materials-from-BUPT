## Sophomore Spring 大二下

### 必修

1. **形式语言与自动机**
    > 应该是这学期最简单的课了
    >
    > 平时把PPT看明白，作业对着答案写明白，期末前刷刷往年题，基本就85+了
    > 
    > 需要注意的是，**实验二**在2022级新增了一部分内容，且实现并不是很容易。
    >
    >22级以前的源码不包括这部分，22级的源码应该还存在部分问题
    >
    > [往年题](形式语言与自动机/往年题)、[实验一](形式语言与自动机/Lab1/)、[实验二](形式语言与自动机/Lab2/)

2. **计算机组成原理**
    > 学的不是很明白，建议找其他人的blog/经验看看
    >
    > 王道考研的书确实不错，建议买回来学/刷
    >
    > 往年题不太好找，但我考完回忆记了一些下来，供小东西们参考：
    >
    > **2024年计组期末（计科）**
    >
    >> 选择题，判断题，概念简答题，计算题，设计分析题
    >>
    >> **选择题**考了浮点数的最大，正数最小，负数知识点，涉及寻址范围，知识点涉及dma传送方式
    >>
    >> **判断题**涉及了平时作业的选择题的选项
    >>
    >> **简答题**第一题是计算CPI之类的
    >>
    >> 第二题是四个概念说明，CPI，SDRAM，第四个是一个M开头的单位（不是MIPS）
    >>
    >> 第三题是把一级catch变成二级cash算提高了多少？
    >> 
    >> 然后是显示器的像素求带宽这种
    >> 
    >> 有一道指令格式的题,有一道磁盘的题（其中一问是这个）
    >>
    >> 最后一题是考研题（王道书上原题），比考研题多了一问，快表+cache
    >
    > 除次之外，论坛上说**23年**考了设计硬布线，片选，内存条，RAM芯片组成
    >
    > 实验也挺难做，[报告在这里](计算机组成原理/lab/)

3. **计算机网络**
    > 也挺难的，期中就考了四五十分，但后半学期认真学 + 期末认真复习 = 90分
    >
    > 依然推荐王道考研书，挺有用的，同样的PPT也很重要
    >
    > 期末个人把程莉老师一学期的课重新听了一遍，收获很多，非常推荐程莉老师
    >
    > 计网每年发的期末样题都是同一套
    >
    > 计网实验挺有意思的，建议认真做，[个人报告+源码](计算机网络/Lab/)

4. **毛泽东思想和中国特色社会主义理论体系概论**
    > 想要好一点的分的话，上课记得去，点名不能缺勤，听讲互动倒是无所谓，特别想卷个95+的话，你就硬卷吧你。
    >
    > 期中展示尽量选新颖点的主题，让老师记住你，印象深刻
    >
    > 最后一节课会**划重点**，**认真听！！！！！！！**，期末对着重点背就行
