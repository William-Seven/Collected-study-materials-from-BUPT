Title: MANTRA: The Manifold Triangulations Assemblage

Authors: Rieck, Bastian, Röell, Ernst, Schmid, Daniel Bīn, Bautista, Rubén Ballester

Keywords: 

Publication date: 2025-06-03

Description:
MANTRA is a dataset consisting of&nbsp;<em>combinatorial triangulations</em>&nbsp;of manifolds. It can be used to create novel algorithms in topological deep learning or debug existing ones. See our&nbsp;<a href="https://openreview.net/pdf?id=X6y5CC44HM" rel="nofollow">ICLR 2025 paper</a> for more details on potential experiments.
