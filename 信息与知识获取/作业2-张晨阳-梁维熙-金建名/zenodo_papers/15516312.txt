The v2.0.0 is the first release with new Edirom Online architecture based on individual repositories for <a href="https://github.com/Edirom/Edirom-Online-Frontend">frontend</a> and <a href="https://github.com/Edirom/Edirom-Online-Backend">backend</a>. The software modules now reside in these remote repositories.&nbsp;


The present release contains the global documentation for the Edirom Online and the build architecture based on Docker. The local setup of an Edirom Online instance - fetching and building of frontend and backend and inclusion of edition packages - can now be done with a few docker compose commands which are documented in the README.&nbsp;
