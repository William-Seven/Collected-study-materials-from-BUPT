Title: Soil biodiversity in the Atacama Desert

Authors: Villegas, Laura, Pettrich, Laura, Acevedo-Trejos, Esteban, Suwanngam, Arunee, Wassey, Nadim, Allende, Miguel, Stoll, Alexandra, Holovachov, Oleksandr, Waldvogel, Ann-Marie, Schiffer, Philipp

Keywords: Biogeography, Nematodes, Environment, Latitude, Atacama

Publication date: 2025-05-26

Description:

This data set contains the fasta sequences of the 18S region of nematodes isolated from the Atacama Desert. This project belong to the CRC1211 "Earth-Evolution at the dry limit" and all files required to reproduce the analysis done in the manuscript titled "Hierarchical Patterns of Soil Biodiversity in the Atacama Desert: Insights Across Biological Scales". All scripts are deposited on github (https://github.com/lauraivillegasr/BiogeographyDesert).
